
# PinData - Keyword Research & Exploration Tool

PinData is a powerful React-based web application designed for discovering, filtering, and organizing keywords for content creators and marketers. It features a modern, clean interface with robust filtering capabilities and mock data to continuously develop and refine your keyword strategy.

## Key Features

- **Keyword Research**: Explore thousands of curated keywords across various niches.
- **Advanced Filtering**: Filter by Categories (Bedroom, Garden, Deck & Patio, Garage, etc.) and Volume ranges.
- **Smart Categorization**: Automatically categorizes keywords into relevant topics like Garden & Landscaping, Deck & Patio, Pool, Garage, and more.
- **Search & Sort**: Powerful search functionality and sorting by Name or Volume.
- **Saved Keywords**: Pin and save your favorite keywords for later access.
- **Board Management**: Organize keywords into specific boards for targeted campaigns.
- **Responsive Design**: Works seamlessly on desktop and tablet devices.
- **Dark Mode**: (Experimental) UI adapts to user preferences.

## Getting Started

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```

## Tech Stack

- **Framework**: React (w/ TypeScript)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Design**: Custom UI components with a focus on usability and aesthetics.
